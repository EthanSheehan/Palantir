**1. Create src/frontend-react/src/cesium/useCesiumZones.ts:**

Direct port of map.js `initOrUpdateZonesPrimitive()` and `getImbalanceColor()`.

- Subscribe to `useSimStore(s => s.zones)` and `useSimStore(s => s.gridVisState)`.
- `primitiveRef = useRef<Cesium.GroundPrimitive | null>(null)` — fill primitive.
- `borderPrimitiveRef = useRef<Cesium.GroundPolylinePrimitive | null>(null)` — border primitive.
- `attrsCacheRef = useRef<Record<string, any>>({})` — cached geometry attributes (same as `zoneAttributesCache` in map.js).

Port `getImbalanceColor(imbalance)` as local function — exact same logic: clamp to [-20, 20], negative = red alpha, positive = blue alpha, zero = transparent.

**CRITICAL pattern — create once, update attributes:**
- First render: create all `GeometryInstance`s and add `GroundPrimitive` + `GroundPolylinePrimitive` to scene.
- Subsequent renders: use `getGeometryInstanceAttributes(zoneId)` + dirty-check `_lastColor` to update colors without recreation (lines 143-167 of map.js).
- Grid visibility: `primitive.show` controlled by `gridVisState` (0=both off, 1=borders only, 2=both on).

Zone ID format: `z_{x_idx}_{y_idx}`.

The `PolygonGeometry` uses corners computed from zone center ± halfWidth/halfHeight (same math as map.js lines 82-95).
Border uses `GroundPolylineGeometry` with positions forming a closed loop (same as map.js lines 112-121).

**3. Update src/frontend-react/src/cesium/CesiumContainer.tsx:**

Wire all four hooks into CesiumContainer. The hooks must be called AFTER the viewer is created. Use a state flag or check `viewerRef.current`:

```typescript
import { useCesiumDrones } from './useCesiumDrones';
import { useCesiumTargets } from './useCesiumTargets';
import { useCesiumZones } from './useCesiumZones';
import { useCesiumFlowLines } from './useCesiumFlowLines';

export function CesiumContainer({ children }: { children?: React.ReactNode }) {
  const containerRef = useRef<HTMLDivElement>(null);
  const viewerRef = useCesiumViewer(containerRef);

  // Entity hooks — all take viewerRef, guard internally
  const { entitiesRef: droneEntitiesRef } = useCesiumDrones(viewerRef);
  const { entitiesRef: targetEntitiesRef } = useCesiumTargets(viewerRef);
  useCesiumZones(viewerRef);
  useCesiumFlowLines(viewerRef);

  return (
    <ViewerContext.Provider value={viewerRef}>
      <div ref={containerRef} style={{ width: '100%', height: '100%', position: 'absolute', top: 0, left: 0 }} />
      {children}
    </ViewerContext.Provider>
  );
}
```

Each hook must guard with `if (!viewerRef.current || viewerRef.current.isDestroyed()) return;` at the top of every useEffect body.
