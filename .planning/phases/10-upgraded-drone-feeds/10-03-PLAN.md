**Rewrite `src/frontend-react/src/overlays/DroneCamPIP.tsx`:**

This component becomes the layout orchestrator. It no longer uses `useDroneCam` — it uses `useSensorCanvas` per slot.

**State model:**
- `camLayout` from Zustand store (SINGLE/PIP/SPLIT/QUAD)
- Local state for per-slot config: `useState<SlotConfig[]>` where:
  ```typescript
  interface SlotConfig {
    droneId: number | null;
    sensorMode: SensorMode;
  }
  ```
- SINGLE: 1 slot (selectedDroneId, default EO_IR)
- PIP: 2 slots (primary=selectedDroneId/EO_IR, secondary=auto-pick next active drone/SAR)
- SPLIT: 2 slots (primary=selectedDroneId/EO_IR, secondary=selectedDroneId/SAR — same drone, different sensors)
- QUAD: 4 slots auto-assigned to the 4 drones with non-IDLE modes (sorted by id), or first 4 if all idle. Default sensor modes: [EO_IR, SAR, EO_IR, SAR]

**Auto-assign QUAD slots logic:**
```typescript
const activeUavs = uavs.filter(u => u.mode !== 'IDLE').slice(0, 4);
if (activeUavs.length < 4) {
  const idle = uavs.filter(u => u.mode === 'IDLE');
  while (activeUavs.length < 4 && idle.length > 0) activeUavs.push(idle.shift()!);
}
```

**Sensor mode selector per slot:**
Each slot has a small sensor mode toggle (4 tiny buttons: EO/SAR/SIG/FUS) rendered in the slot's top-right corner. When SIGINT is selected, the canvas is hidden and SigintDisplay is shown instead. Otherwise canvas + useSensorCanvas renders.

**Slot component — `CamSlot`** (defined inside DroneCamPIP or as a local component):
```typescript
function CamSlot({ droneId, sensorMode, onSensorModeChange, width, height }: {
  droneId: number | null;
  sensorMode: SensorMode;
  onSensorModeChange: (mode: SensorMode) => void;
  width: number;
  height: number;
}) {
  const canvasRef = useRef<HTMLCanvasElement>(null);
  useSensorCanvas(droneId, sensorMode, canvasRef);
  const drone = useSimStore(s => droneId != null ? s.uavs.find(u => u.id === droneId) ?? null : null);
  const targets = useSimStore(s => s.targets);

  return (
    <div style={{ position: 'relative', width, height }}>
      {sensorMode === 'SIGINT' ? (
        <SigintDisplay droneId={droneId} width={width} height={height} />
      ) : (
        <canvas ref={canvasRef} style={{ display: 'block', width, height }} />
      )}
      <SensorHUD drone={drone} targets={targets} sensorMode={sensorMode} />
      {/* Mini sensor mode toggle — top right of slot */}
      <div style={{ position: 'absolute', top: 2, right: 2, display: 'flex', gap: 1 }}>
        {(['EO_IR', 'SAR', 'SIGINT', 'FUSION'] as SensorMode[]).map(m => (
          <button key={m} onClick={() => onSensorModeChange(m)}
            style={{
              background: sensorMode === m ? 'rgba(45,114,210,0.6)' : 'rgba(0,0,0,0.5)',
              border: '1px solid rgba(255,255,255,0.2)',
              color: '#ccc', fontSize: 8, fontFamily: 'monospace',
              padding: '1px 3px', cursor: 'pointer', lineHeight: 1,
            }}>
            {m === 'EO_IR' ? 'EO' : m === 'SIGINT' ? 'SIG' : m === 'FUSION' ? 'FUS' : m}
          </button>
        ))}
      </div>
    </div>
  );
}
```
