   - `target_id: int`

   - `target_id: int`
   - `mode: str` (always `"SUPPORT"`)
   - `reason: str`

4. `class SwarmCoordinator`:
   - `__init__(self, min_idle_count: int = 3)` — stores min_idle_count, creates `_active_tasks: dict[int, SwarmTask] = {}`
   - `evaluate_and_assign(self, targets: list, uavs: list) -> list[TaskingOrder]`:
     a. Update _active_tasks from current UAV/target state (remove tasks for targets no longer DETECTED/CLASSIFIED, update sensor_coverage)
     b. Score each DETECTED/CLASSIFIED target: `_priority_score(target)`, skip if `_sensor_gap(target)` empty
     c. Sort scored targets descending
     d. Count idle UAVs: `sum(1 for u in uavs if u.mode in ("IDLE", "SEARCH"))`
     e. For each target, for each gap sensor type: if idle_count > min_idle_count, call `_find_nearest(uavs, target, sensor_type, already_assigned)`, create TaskingOrder, decrement idle_count, add uav.id to already_assigned set
     f. Update _active_tasks with new SwarmTask entries for any orders issued
     g. Return orders list
   - `_sensor_gap(self, target) -> list[str]`: return sensor types from SENSOR_TYPES not in `{c.sensor_type for c in target.sensor_contributions}`
   - `_priority_score(self, target) -> float`: `THREAT_WEIGHTS.get(target.type, 0.5) * (1.0 - target.fused_confidence)` where THREAT_WEIGHTS = {"SAM": 1.0, "TEL": 0.9, "CP": 0.7, "TRUCK": 0.5, "MANPADS": 0.8, "RADAR": 0.9, "C2_NODE": 0.6, "LOGISTICS": 0.3, "ARTILLERY": 0.8, "APC": 0.6}
   - `_find_nearest(self, uavs, target, sensor_type: str, exclude: set[int] | None = None)`: filter by mode in ("IDLE", "SEARCH"), sensor_type in sensors, id not in exclude; return min by `math.hypot(u.x - target.x, u.y - target.y)` or None
   - `get_active_tasks(self) -> dict[int, SwarmTask]`: return `dict(self._active_tasks)`

Use `from __future__ import annotations` at top. Import `math`, `dataclasses.dataclass`, `typing.Dict, List, Optional`.

THREAT_WEIGHTS as module-level constant dict.
