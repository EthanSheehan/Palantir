**useCoverageLayer.ts** (~130 lines):

1. Signature: `export function useCoverageLayer(viewerRef: React.RefObject<Cesium.Viewer | null>)`
2. Use `useRef<Cesium.Primitive | null>(null)` for the primitive and `useRef<number>(0)` for tracking UAV count.
3. Subscribe to `useSimStore` in a single `useEffect([viewerRef])`.
4. On each store update:
   - Read `state.layerVisibility['coverage'] ?? false` for visibility.
   - If not visible: set `primitiveRef.current.show = false` if primitive exists, then return.
   - If visible and primitive exists: set `show = true`.
   - Build/rebuild `GroundPrimitive` when UAV count changes (not every tick). Track count in countRef.
   - For each UAV, create a `Cesium.GeometryInstance` with `Cesium.EllipseGeometry`:
     - `center`: `Cesium.Cartesian3.fromDegrees(uav.lon, uav.lat)`
     - `semiMajorAxis`: `SENSOR_RANGE_KM * 1000` (import from constants)
     - `semiMinorAxis`: same
     - `height`: 0
   - Color by first sensor: EO_IR = `rgba(6, 182, 212, 0.15)` (cyan), SAR = `rgba(234, 179, 8, 0.15)` (yellow), SIGINT = `rgba(168, 85, 247, 0.15)` (purple). Use `Cesium.ColorGeometryInstanceAttribute.fromColor()`.
   - Use `Cesium.PerInstanceColorAppearance({ flat: true, translucent: true })`.
   - When rebuilding: remove old primitive first, then add new one.
5. Cleanup: unsub + remove primitive if viewer alive.

**useThreatLayer.ts** (~130 lines):

1. Same signature pattern as useCoverageLayer.
2. Subscribe to store, check `state.layerVisibility['threat'] ?? false`.
3. For each target where `THREAT_RING_TYPES.has(target.type)` and `target.detected`:
   - Create `Cesium.GeometryInstance` with `Cesium.EllipseGeometry`:
     - `center`: `Cesium.Cartesian3.fromDegrees(target.lon, target.lat)`
     - `semiMajorAxis`: `(target.threat_range_km ?? 5) * 1000`
     - `semiMinorAxis`: same
     - `height`: 0
   - SAM color: `rgba(255, 68, 68, 0.2)` fill. MANPADS: `rgba(236, 72, 153, 0.2)`.
4. Rebuild when detected threat target count changes (track in ref).
5. Use `primitive.show = false/true` for visibility toggle (not remove/add).
6. Cleanup same pattern as coverage.

Both hooks guard: `if (!viewer || viewer.isDestroyed()) return;` at top of subscriber.

**useSwarmLayer.ts** (~100 lines):

1. Same hook pattern. Check `state.layerVisibility['swarm'] ?? false`.
2. Use `viewer.entities` (not GroundPrimitive) since line count is small.
3. Track entities in `entitiesRef = useRef<Cesium.Entity[]>([])`.
4. For each UAV in FOLLOW/PAINT/INTERCEPT/SUPPORT mode with `tracked_target_ids.length > 0`:
   - For the primary target (`primary_target_id` or `tracked_target_ids[0]`):
     - Find target in `state.targets` by id.
     - If found, draw a dashed polyline from UAV position to target position.
     - Use `Cesium.PolylineDashMaterialProperty({ color: Cesium.Color.CYAN.withAlpha(0.6), dashLength: 16 })`.
     - Set `entity.polyline.width = 2`.
5. On each update when visible: remove all old entities, rebuild. Count is small so this is fine.
6. When not visible: remove all entities from viewer.
7. Cleanup: remove all entities.

**useTerrainLayer.ts** (~100 lines):

1. Same hook pattern. Check `state.layerVisibility['terrain'] ?? false`.
2. When terrain becomes visible:
   - Set `viewer.scene.globe.terrainExaggeration = 2.0` (if property exists — guard with `if ('terrainExaggeration' in viewer.scene.globe)`).

     - Width: 1.
   - Track entities in ref like swarm layer.
3. When terrain becomes hidden:
   - Reset `viewer.scene.globe.terrainExaggeration = 1.0`.
   - Remove all LOS line entities.
4. CRITICAL: cleanup function MUST reset `terrainExaggeration = 1.0` to prevent bleed to other modes.

All three hooks: guard `if (!viewer || viewer.isDestroyed()) return;` in subscriber.
