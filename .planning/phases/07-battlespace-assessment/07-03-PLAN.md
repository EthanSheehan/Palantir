    3. Use the subscribe pattern from useCesiumFlowLines (NOT React re-render):
       ```typescript
       const entitiesRef = useRef<Cesium.Entity[]>([]);

       useEffect(() => {
         const unsub = useSimStore.subscribe((state) => {
           const viewer = viewerRef.current;
           if (!viewer || viewer.isDestroyed()) return;
           const assessment = state.assessment;
           if (!assessment) return;

           // Remove all previous entities
           entitiesRef.current.forEach(e => viewer.entities.remove(e));
           entitiesRef.current = [];

           // 1. Render cluster hull polygons
           // 2. Render SAM engagement envelopes
           // 3. Render movement corridor polylines

           viewer.scene.requestRender();
         });
         return () => unsub();
       }, [viewerRef]);
       ```

    4. Cluster hull polygons: For each cluster in assessment.clusters:
       - Color mapping: SAM_BATTERY -> Color.RED, AD_NETWORK -> Color.ORANGE, CONVOY -> Color.BLUE, CP_COMPLEX -> Color.PURPLE, MIXED -> Color.GRAY
       - If hull_points.length >= 3: create polygon entity with PolygonHierarchy

         const positions = Cesium.Cartesian3.fromDegreesArray(

             height: 0,
           },

    5. SAM engagement envelopes: Read targets from state.targets, filter for SAM/RADAR/MANPADS types where state !== 'UNDETECTED' and threat_range_km is not null:
       ```typescript
       const samTargets = state.targets.filter(
         t => ['SAM', 'RADAR', 'MANPADS'].includes(t.type)
           && t.state !== 'UNDETECTED'
           && t.threat_range_km != null
       );
       for (const t of samTargets) {
         const radius = (t.threat_range_km ?? 0) * 1000; // km to meters
         if (radius <= 0) continue;
         const ring = viewer.entities.add({
           position: Cesium.Cartesian3.fromDegrees(t.lon, t.lat),
           ellipse: {
             semiMajorAxis: radius,
             semiMinorAxis: radius,
             material: Cesium.Color.RED.withAlpha(0.1),
             outline: true,
             outlineColor: Cesium.Color.RED.withAlpha(0.6),
             outlineWidth: 1.5,
             height: 0,
           },
         });
         entitiesRef.current.push(ring);
       }
       ```
