    2. Module-level constants:
       - CLUSTER_RADIUS_DEG = 0.135 (15km / 111km per degree)
       - MIN_CLUSTER_SIZE = 2
       - POSITION_HISTORY_MIN = 10 (minimum entries for corridor)
       - CORRIDOR_MIN_MOVEMENT_DEG = 0.005
       - CLUSTER_AFFINITY = {"SAM": "AD_NETWORK", "RADAR": "AD_NETWORK", "MANPADS": "AD_NETWORK", "TEL": "SAM_BATTERY", "CP": "CP_COMPLEX", "C2_NODE": "CP_COMPLEX", "TRUCK": "CONVOY", "LOGISTICS": "CONVOY", "APC": "CONVOY", "ARTILLERY": "CONVOY"}

    3. BattlespaceAssessor class with:
       - assess(targets: list[dict], uavs: list[dict], zones: list[dict]) -> AssessmentResult
         - Calls _cluster_targets, _identify_coverage_gaps, _score_zone_threats, _detect_movement_corridors
         - Returns frozen AssessmentResult with time.time() as assessed_at
       - _cluster_targets(targets): Greedy distance-based clustering on detected targets only.
         - Filter state != UNDETECTED
         - For each unvisited target, find all neighbors within CLUSTER_RADIUS_DEG using math.hypot
         - If len(neighbors) >= MIN_CLUSTER_SIZE, form cluster
         - Cluster type = majority vote from CLUSTER_AFFINITY, fallback "MIXED"
         - Cluster ID = "CLU-" + "-".join(sorted(str(id) for id in member_ids)) for stability
         - Compute convex hull via _compute_convex_hull
         - threat_score = mean of fused_confidence values
       - _identify_coverage_gaps(zones, uavs): Return CoverageGap for zones where uav_count==0 AND not covered by any UAV in SEARCH/OVERWATCH/REPOSITIONING mode at same (x_idx, y_idx). Compute covered set from uavs by finding nearest zone per UAV.
       - _score_zone_threats(zones, targets): For each detected target, assign to nearest zone by lon/lat distance, sum fused_confidence per zone capped at 1.0. Return dict[(x_idx, y_idx), score].
       - _detect_movement_corridors(targets): For each target dict that has 'position_history' key with len >= POSITION_HISTORY_MIN, compute total displacement. If > CORRIDOR_MIN_MOVEMENT_DEG, create MovementCorridor with waypoints sampled at every 5th entry.

    4. _compute_convex_hull(points) -> tuple[tuple[float, float], ...]:
       - Jarvis march (gift-wrapping), handle 0/1/2 point edge cases.
